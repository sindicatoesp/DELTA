- id: P0-27
  title: "Add NOTICE file — Apache 2.0 Section 4(d) attribution requirement"
  file: null
  line: null
  root_cause: >
    A NOTICE file does not exist in the repository. Apache 2.0 Section 4(d)
    requires that any redistribution include a NOTICE file attributing the
    original authors when one is provided by the Licensor. The item notes specify
    the primary attribution line: "DELTA Resilience Platform — Copyright 2026
    United Nations Office for Disaster Risk Reduction (UNDRR)". Third-party
    bundled Apache 2.0 dependencies with their own NOTICE content should also be
    listed. Without this file the project is technically non-compliant with its
    own license terms for redistributions.
  fix_complexity: Simple
  requires_spec: false
  requires_test: false
  fix_summary: >
    Create a NOTICE file with the primary attribution line for UNDRR/DELTA;
    audit bundled Apache 2.0 dependencies for their own NOTICE content and
    include any required attributions.
  antipattern: null
---
