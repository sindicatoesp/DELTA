---
- id: P0-12
  title: "Remove secret logging in env.ts"
  file: app/utils/env.ts
  line: 20
  root_cause: >
    app/utils/env.ts line 20 contains `console.log("Loading env file", type)`
    which logs the .env file type name (e.g. "test" or "") at startup using
    unstructured console.log. The current code does NOT log actual env var
    key/value pairs; the copilot-instructions description ("logs all environment
    variable values at startup, including secrets") describes a former or intended
    future risk rather than the present code. However, the console.log at line 20
    and console.warn at line 41 still violate the project's structured-logger
    convention (AP-04) and should be replaced with the logger from
    app/utils/logger.ts or removed entirely. The env.ts startup logs are
    loaded before the structured logger is initialised, so removal is safer.
  fix_complexity: Trivial
  requires_spec: false
  requires_test: false
  fix_summary: >
    Remove or replace the console.log at line 20 and console.warn at line 41
    in env.ts with the structured logger or silence them entirely.
  antipattern: AP-04
---
