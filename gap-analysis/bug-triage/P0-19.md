- id: P0-19
  title: "Fix build_binary.sh — build failure must be fatal"
  file: scripts/build_binary.sh
  line: 33
  root_cause: >
    scripts/build_binary.sh opens with `set -e` (line 1) which would normally
    abort on any non-zero exit. However, lines 33-35 wrap the build step in an
    explicit error-swallowing block:
      if ! yarn build; then
        echo "WARNING: yarn build failed, continuing anyway..."
      fi
    This explicitly catches a build failure and continues script execution,
    packaging whatever incomplete or previous build artifacts exist in the build/
    directory. The resulting binary may be silently broken. The set -e guarantee
    is nullified for the most critical step.
  fix_complexity: Trivial
  requires_spec: false
  requires_test: false
  fix_summary: >
    Replace the if/fi block at lines 33-35 with a plain `yarn build` call;
    set -e will abort the script if the build fails.
  antipattern: AP-12
---
